# Sistema-de-Gesti-n-de-proyectos-clientes-para-dise-adores-freelance
El presente proyecto implementa un sistema de gestión de proyectos y clientes para un diseñador freelance, con persistencia de datos en archivos CSV. El objetivo general es ofrecer una herramienta de consola que permita registrar, consultar, modificar y eliminar proyectos de diseño,.
